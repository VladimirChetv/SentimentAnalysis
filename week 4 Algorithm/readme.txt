Демо анализа тональности отзывов
1. Установить необходимые библиотеки:
$ pip install flask
2. Запустить демо:
$ python demo.py
3. Дождаться, когда будет готов классификатор. (~70  сек.)
4. Открыть в браузере страницу:
http://127.0.0.1:8080/
